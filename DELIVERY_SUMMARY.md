## 🚀 THREE-STEP LAUNCH

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Configure Firebase
1. Create Firebase project at https://console.firebase.google.com
2. Get your Firebase credentials
3. Update `.env` file with credentials
4. Create 4 Firestore collections: contacts, subscribers, appointments, blogs

### Step 3: Run Application
```bash
npm run dev
```

**That's it! Open http://localhost:5173 in your browser** 🎉

## 🆘 Troubleshooting

### Issue: npm install fails
**Solution**: Check Node.js is installed: `node --version`

### Issue: Forms not saving
**Solution**: Verify Firestore collections are created

### Issue: Blank page
**Solution**: Check .env has correct Firebase config

### Issue: Port already in use
**Solution**: `npm run dev -- --port 3000`

More help in QUICKSTART.md or README.md
